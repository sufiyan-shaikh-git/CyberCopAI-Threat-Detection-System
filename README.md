ğŸ§¬ How it Works (The Logic)
CyberCop extracts critical features from the PE headers of an uploaded file, such as:

Entropy: To detect packed or encrypted malicious code.

DLL Characteristics: To check for security flags like ASLR and DEP.

Machine ID: To identify the target architecture.

The extracted data is then passed to the pre-trained malware_model.pkl, which classifies the file as either Legitimate or Malicious within seconds.
